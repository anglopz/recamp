# Archive Directory

This directory contains legacy code and temporary files from the migration process.

## Structure

- `legacy/` - Original JavaScript/EJS codebase (kept for reference)
  - `app.js` - Original Express app entry point
  - `controllers/` - Original controllers
  - `models/` - Original Mongoose models
  - `routes/` - Original route definitions
  - `views/` - Original EJS templates
  - `public/` - Original static assets
  - `utils/` - Original utility functions
  - `middleware.js` - Original middleware
  - `schemas.js` - Original Joi schemas
  - `cloudinary/` - Original Cloudinary config
  - `seeds/` - Database seed files

## Note

These files are archived and no longer used in the application. The modernized codebase is in `src/backend/` and `src/frontend/`.
